# 🧠 SoulSync

SoulSync is a **mental wellness and emotional connection platform** designed to help users reflect on their emotions, track their mood, and maintain better mental health.

---

## 🚀 Features

- Mood tracking
- Emotional journaling
- Personalized insights
- Clean mobile UI
- Mental wellness focus

---

## 🛠 Tech Stack

- React Native
- Expo
- TypeScript
- JavaScript

---

## 📂 Project Structure

app/ → Screens and navigation  
components/ → Reusable UI components  
hooks/ → Custom React hooks  
constants/ → Global constants  
assets/ → Images and icons
